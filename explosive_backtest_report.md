## 🔍 문제점 분석

### 1. 과도한 진입 기준 (70점)
```
3년간 총 4회 매매 → 연평균 1.3회
→ 기회비용 손실 심각
→ 시장 상승기 수익 참여 못함
```

### 2. 보유만기 청산의 문제
- 바이오노트: -9.61% (보유만기 청산, 63거래일)
- 3개월 보유 제한으로 인한 강제 청산

### 3. 낮은 승률의 심리적 영향
- 25% 승률 → 연속 손실 시 전략 신뢰도 하띱

## ⚠️ 주의사항

1. **과적합(Overfitting) 주의**: 백테스트 기간에 최적화된 파라미터는 미래에 통하지 않을 수 있음

2. **거래비용 미반영**: 실제 수익률은 수수료/세금으로 0.5~1% 추가 차감

3. **유동성 리스크**: 소형주 급등 시 매도 물량 부족 가능성

4. **시장 체제 변화**: 박스권/하락장에서 전략 효과 감소
