## 전략 1: 이동평균선 크로스오버 (Moving Average Crossover)

### 전략 개요
| 항목 | 내용 |
|:---|:---|
| **코드명** | `MA_CROSS` |
| **유형** | 추세 추종 |
| **시간대** | 일봉/4시간봉 |
| **승률** | 35~45% |
| **R:R** | 1:2 ~ 1:3 |

### 진입 조건
```
[골든크로스 매수]
- 50일 이동평균 > 200일 이동평균 (정배열)
- 단기선(5일/10일)이 장기선(20일/50일) 상향 돌파
- 거래량 20일 평균 대비 1.2배 이상

[데드크로스 매도/관망]
- 50일 이동평균 < 200일 이동평균 (역배열)
- 포지션 청산 또는 공매도
```

### 손절/익절
```python
# 손절가
stop_loss = min(20일선, 50일선)  # 근접 이동평균

# 익절가 (R:R 1:3)
entry_price = 현재가
target_price = entry_price + (entry_price - stop_loss) * 3

# 트레일링 스톱
if 수익률 > 10%:
    stop_loss = entry_price  # 본전 보장
```

### 장단점
| 장점 | 단점 |
|:---|:---|
| 구현 간단 | 늦은 진입 (추세 확인 후) |
| 명확한 신호 | 횡보장에서 위스프(whiplash) |
| 강한 추세에서 수익 | 약한 추세에서 손실 |

## 전략 2: ATR 채널 브레이크아웃 (ATR Channel Breakout)

### 전략 개요
| 항목 | 내용 |
|:---|:---|
| **코드명** | `ATR_BREAK` |
| **유형** | 변동성 돌파 |
| **시간대** | 일봉 |
| **승률** | 40~50% |
| **R:R** | 1:2.5 ~ 1:4 |

### 진입 조건
```
[매수 진입]
- 종가 > 350일 이동평균 + (7 × ATR(14))
- 상향 채널 돌파 확인

[매도 진입/청산]
- 종가 < 350일 이동평균 - (3 × ATR(14))
- 하향 채널 돌파 또는 350일선 하향 이탈
```

### 채널 계산
```python
# 상단 채널 (매수 트리거)
upper_channel = MA(350) + 7 * ATR(14)

# 하단 채널 (매도 트리거)
lower_channel = MA(350) - 3 * ATR(14)

# 중간선 (추세 확인)
middle_line = MA(350)
```

### 포지션 사이징
```python
# ATR 기반 포지션 크기
risk_per_trade = 계좌잔고 * 0.02  # 2% 리스크
atr = ATR(14)
shares = risk_per_trade / (2 * atr)  # 2ATR 손절
```

### 장단점
| 장점 | 단점 |
|:---|:---|
| 변동성 고려 | 복잡한 계산 |
| 추세 강도 반영 | 늦은 신호 가능 |
| 명확한 손절 기준 | 고변동성 시 거짓 신호 |

## 전략 3: 볼린저 밴드 + MFI 추세 추종

### 전략 개요
| 항목 | 내용 |
|:---|:---|
| **코드명** | `BB_MFI` |
| **유형** | 추세 확인 + 과매수/과매도 |
| **시간대** | 일봉/4시간봉 |
| **승률** | 45~55% |
| **R:R** | 1:2 ~ 1:2.5 |

### 지표 설정
```python
# 볼린저 밴드
BB_period = 20
BB_std = 2.5

# MFI (Money Flow Index)
MFI_period = 14

# %B (볼린저 밴드 위치)
%B = (종가 - 하단) / (상단 - 하단)
```

### 진입 조건
```
[매수 진입 - 추세 확인]
- %B > 0.8 (상단 근접)
- MFI > 80 (강한 매수세)
- 20일선 위 (상승 추세)

[매도 진입 - 추세 전환]
- %B < 0.2 (하단 근접)
- MFI < 20 (강한 매도세)
- 20일선 아래 (하락 추세)

[청산]
- 중심선(20일선) 돌파
- 반대 신호 발생
```

### 장단점
| 장점 | 단점 |
|:---|:---|
| 거래량 + 가격 동시 고려 | 복잡한 다중 지표 |
| 과매수/과매도 구분 | MFI 왜곡 가능 |
| 추세 강도 측정 | 채널 돌파 전 눌림목 손실 |

## 전략 4: 돈키안 채널 브레이크아웃 (Donchian Channel)

### 전략 개요
| 항목 | 내용 |
|:---|:---|
| **코드명** | `DONCHIAN` |
| **유형** | 고점/저점 돌파 |
| **시간대** | 일봉/주봉 |
| **승률** | 35~40% |
| **R:R** | 1:3 ~ 1:5 |

### 진입 조건
```
[매수 진입]
- 종가 > 최근 N일 최고가 (상단 돌파)
- N = 20 (단기) 또는 55 (중기)
- 거래량 증가 확인

[매도 진입/청산]
- 종가 < 최근 N일 최저가 (하단 돌파)
- 또는 최근 N/2일 최저가 하향 돌파 (시간 청산)
```

### 채널 계산
```python
# 돈키안 상단 (저항선)
upper = max(고가[-N:])

# 돈키안 하단 (지지선)
lower = min(저가[-N:])

# 돈키안 중심
middle = (upper + lower) / 2
```

### 특징
- **거북이 거래법(Turtle Trading)**의 핵심 전략
- 강한 추세에서 큰 수익
- 횡보장에서 연속 손실 (위스프)

### 장단점
| 장점 | 단점 |
|:---|:---|
| 구현 간단 | 횡보장에서 손실 누적 |
| 강한 추세 포착 | 늦은 진입 |
| 명확한 신호 | 승률 낮음 (40% 미만) |

## 전략 5: 듀얼 HMA 모멘텀 브레이크아웃

### 전략 개요
| 항목 | 내용 |
|:---|:---|
| **코드명** | `DUAL_HMA` |
| **유형** | 고급 추세 추종 |
| **시간대** | 일봉/4시간봉 |
| **승률** | 40~50% |
| **R:R** | 1:2 ~ 1:3 |

### 진입 조건
```
[매수 진입 - 모든 조건 충족]
1. HMA(20) > HMA(200)  # 상승 추세
2. SMA(5) > HMA(200)   # 단기 상승
3. 종가 > 시가 AND 종가 > 전일고가  # 강한 양봉
4. (종가 - HMA(20)) > ATR(14) × 0.5  # 충분한 거리
5. 거래량 > 거래량SMA  # 거래량 확인
6. 30 < RSI(14) < 70   # 과매수/과매도 아님
7. MACD > 시그널선     # 모멘텀 확인

[매도 진입/청산]
- HMA(20) < HMA(200) (추세 전환)
- 또는 RSI > 70 (과매수)
- 또는 MACD < 시그널선 (모멘텀 약화)
```

### 손절/익절
```python
# ATR 기반 동적 손절
stop_loss = entry_price - 2 * ATR(14)

# R-배수 목표가
target_2R = entry_price + 2 * (entry_price - stop_loss)  # 2R 익절
target_3R = entry_price + 3 * (entry_price - stop_loss)  # 3R 익절

# 트레일링 스톱
if price >= target_2R:
    stop_loss = entry_price  # 본전 보장
```

### 장단점
| 장점 | 단점 |
|:---|:---|
| 다중 필터로 신뢰도 ↑ | 복잡한 진입 조건 |
| HMA로 빠른 반응 | 진입 기회 감소 |
| 명확한 R:R 관리 | 놓치는 추세 다수 |

## 전략 6: 15분봉 120선 전환존 (전황 트레이더)

### 전략 개요
| 항목 | 내용 |
|:---|:---|
| **코드명** | `M15_MA120` |
| **유형** | 단기 추세 추종 |
| **시간대** | 15분봉 |
| **승률** | 50~60% |
| **R:R** | 1:1.5 ~ 1:2 |

### 진입 조건
```
[매수 타점 - 전환존]
- 15분봉 120선 근접 (닿거나 살짝 아래)
- 60선 > 120선 (정배열)
- 변동폭: 대형주 2~4%, 주도주 3~8%
- 고점에서 4~5% 눌림 후 반등 확인

[손절]
- 120선 하향 돌파 시 즉시 손절
```

### 익절 전략
```
[익절 타이밍]
- 당일 매도하지 않음
- 다음날 오전 슈팅(급등) 시 매도
- 목표 수익: 5~10%
- 손절 기준 아니면 무조건 보유 다음날
```

### 장단점
| 장점 | 단점 |
|:---|:---|
| 단기 고승률 | 모니터링 필요 |
| 명확한 손절 | 변동성 큰 종목 필수 |
| 리스크 작음 | 대형주 위주 |

## 전략 7: RSI(2) 평균회귀 + 200일선 추세필터

### 전략 개요
| 항목 | 내용 |
|:---|:---|
| **코드명** | `RSI2_MA200` |
| **유형** | 평균회귀 + 추세필터 |
| **시간대** | 일봉 |
| **승률** | 55~65% |
| **R:R** | 1:1 ~ 1:1.5 |

### 진입 조건
```
[매수 진입 - 반등 노림]
- RSI(2) < 25  # 극단적 과매도
- 종가 > 200일 이동평균  # 장기 상승 추세
- 상승 추세 내 단기 눌림목
```

### 청산 조건
```
[익절]
- 목표가: 최근 2거래일 최고가

[손절/시간청산]
- 5거래일 경과 시 무조건 청산
- 고정 손절 없음 (시간 청산으로 리스크 관리)
```

### 특징
- **평균회귀 성격**: 과매도 반등 노림
- **추세필터**: 상승 추세 내에서만 진입
- **고승률, 낮은 R:R**

### 장단점
| 장점 | 단점 |
|:---|:---|
| 높은 승률 (60%+) | 낮은 R:R |
| 명확한 목표가 | 5일 청산으로 수익 제한 |
| 추세 내 안전한 진입 | 약세장 위험 |
