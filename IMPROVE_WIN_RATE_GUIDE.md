## 1️⃣ 리스크/리워드 비율 최적화

### 핵심 원리
| R:R 비율 | 필요 승률 (손익분기) | 실제 적용 |
|:---|:---|:---|
| **1:3** | 25%만 승리필요 | 최적의 비율 |
| 1:2 | 33.3% 승리 필요 | 보수적 전략 |
| 1:1.5 | 40% 승리 필요 | 단타용 |

> 💡 **핵심**: 낮은 승률로도 수익 가능하도록 R:R을 1:2 이상 유지

### 적용 방법
```
진입가: 10,000원
손절가: 9,800원 (-2%, 200원 리스크)
목표가: 10,600원 (+6%, 600원 리워드)
→ R:R = 1:3 (600/200)
→ 승률 30%만으로도 수익
```

## 2️⃣ 다중 지표 조합 전략

### 추천 조합 (KAGGRESSIVE 개선용)

#### A. 추세 확인
| 지표 | 설정 | 신호 |
|:---|:---|:---|
| 이동평균선 | 5일/20일/60일 | 골든크로스 진입 |
| MACD | 12, 26, 9 | MACD > Signal |
| ADX | 14 | ADX > 25 (강한 추세) |

#### B. 과매수/과매도
| 지표 | 설정 | 진입 조건 |
|:---|:---|:---|
| RSI | 14 | 40~60 (중립구간 상승) |
| 스토캐스틱 | 14, 3, 3 | %K > %D 교차 |
| 볼린저 밴드 | 20, 2 | 하단→중단 돌파 |

#### C. 거래량 확인
- 거래량 20일 평균 대비 **150% 이상**
- OBV (On Balance Volume) 상승세


### 피할 때 (NO 진입)
```

❌ 연휴 직전/직후

```

## 4️⃣ 포지션 사이징 (Position Sizing)

### 켈리 공식 기반
```python
# 켈리 비율 = (승률 × 평균수익 - 패배율 × 평균손실) / 평균수익
# 보수적: 켈리 비율의 1/4 ~ 1/2 사용

예시:
- 승률 40%
- 평균 수익 +8%
- 평균 손실 -3%

켈리 = (0.4 × 0.08 - 0.6 × 0.03) / 0.08 = 0.175 (17.5%)
보수적 = 17.5% / 4 = 4.4% (1거래당 자본의 4.4%)
```

### KAGGRESSIVE 적용
| 점수 | 포지션 크기 |
|:---|:---|
| 70~75점 | 자본의 2% |
| 76~80점 | 자본의 3% |
| 81~85점 | 자본의 4% |
| 86점+ | 자본의 5% |

## 5️⃣ 심리적 지표 활용

### 공포/탐욕 지수 (Fear & Greed Index)
```
0~20 (극도 공포): 반등 매수 기회 ⚠️ 주의
21~40 (공포): 매수 적기
41~60 (중립): 정상 거래
61~80 (탐욕): 익절 고려
81~100 (극도 탐욕): 매도 또는 관망
```

### VIX (변동성 지수)
```
VIX < 15: 안정적, 모멘텀 전략 유리
VIX 15~20: 정상
VIX 20~25: 주의, 포지션 축소
VIX > 25: 회피 또는 역추세 전략
```

## 6️⃣ KAGGRESSIVE 개선 제안

### 현재 상태 분석
```
현재: 65점 기준, 7일 보유, 승률 25%, 수익률 +5.78%
문제: 승률 낮음 (25%), 패배 거래 다수
```

### 개선안 A: 다중 필터 추가
```python
진입 조건 = (
    KAGGRESSIVE 점수 >= 70  # 기존 65 → 70 상향
    AND RSI(14) > 40        # 과매도 탈출
    AND RSI(14) < 65        # 과매수 미진입
    AND 거래량 > 20일평균 × 1.5
    AND 종가 > 5일 이동평균
    AND 시장 VIX < 20
)
```

### 개선안 B: 동적 손절/익절
```python
# 추세 강도에 따른 동적 목표가
if ADX > 30:      # 강한 추세
    target_ratio = 1:3.5
elif ADX > 20:    # 보통 추세
    target_ratio = 1:2.5
else:             # 약한 추세
    target_ratio = 1:1.5

# 트레일링 스톱
if 수익률 > 5%:
    stop_loss = 진입가 + (현재가 - 진입가) × 0.5  # 본전 + 50% 수익 보장
```

### 개선안 C: 섹터 회전 분석
```
상승 섹터의 종목만 선별
- 반도체 강세 → 한미반도체, 넥스틴 등
- 바이오 강세 → 관련 종목 필터링
- 강세 섹터 종목에 가중치 +10점
```

## 7️⃣ 검증된 전략 사례

### Bull Flag 패턴
```
특징:
- 강한 상승 (깃발봉) + 저거래량 조정
- 조정폭 38.2%~50% 피본acci
- 돌파 시 거래량 급증

진입: 돌파 시
손절: 조정 저점 아래
목표: 깃발봉 높이만큼
승률: 60~70% (연구 결과)
```

### 모멘텀 브레이크아웃
```
특징:
- 20일 고점 돌파
- 거래량 2배 이상
- RSI 50~70 구간

진입: 돌파 확인 후
손절: 돌파 저점
목표: 1:3 R:R
승률: 55~65%
```
