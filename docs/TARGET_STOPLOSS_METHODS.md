# 목표가/손절가 설정 방법 연구 보고서

## Executive Summary

본 보고서는 주식/암호화폐 트레이딩에서 사용되는 다양한 목표가(Target Price) 및 손절가(Stop-Loss) 설정 방법론을 체계적으로 정리합니다.

---

## 1. 기본 원칙: 손익비(Risk-Reward Ratio)

### 1.1 손익비의 중요성

| 손익비 | 승률 요구 | 특징 |
|:---|:---:|:---|
| 1:1 | 50%+ | 스캘핑, 높은 승률 필요 |
| 1:2 | 33%+ | 최소 권장 비율 |
| 1:3 | 25%+ | 프로 트레이더 표준 |
| 1:5 | 17%+ | 추세 추종 전략 |

### 1.2 계산 공식

    Risk = 진입가 - 손절가
    Reward = 목표가 - 진입가
    Risk-Reward Ratio = Reward / Risk

---

## 2. 손절가(Stop-Loss) 설정 방법

### 2.1 ATR 기반 손절가 [추천]

**공식:**

    손절가 = 진입가 - (ATR x 승수)
    - 보수적: ATR x 1.5
    - 표준: ATR x 2.0
    - 공격적: ATR x 3.0

**장점**: 변동성에 따른 자동 조절, 감정적 결정 제거

### 2.2 피볼나치 되돌림 기반

| 레벨 | 용도 | 특징 |
|:---|:---|:---|
| 38.2% | 단기 스윙 | 타이트한 손절, 높은 승률 |
| 50% | 중기 포지션 | 균형 잡힌 리스크 |
| 61.8% | 장기 추세 | 넓은 손절, 큰 수익 |
| 78.6% | 마지노선 | 마지막 방어선 |

### 2.3 ICT/SMC 구조 기반

#### Fair Value Gap (FVG) 기준
- Bullish FVG: 진입=CE(50%), 손절=FVG 하단 아래
- Bearish FVG: 진입=CE(50%), 손절=FVG 상단 위

#### Order Block (OB) 기준
- Bullish OB: 진입=OB 상단 돌파, 손절=OB 하단 이탈
- Bearish OB: 진입=OB 하단 이탈, 손절=OB 상단 돌파

#### Market Structure 기준
- Bullish BOS: 이전 Swing Low 아래
- Bearish BOS: 이전 Swing High 위

## 3. 목표가(Target Price) 설정 방법

### 3.1 피볼나치 확장 (Fibonacci Extension)

| 레벨 | 목표가 특성 | 용도 |
|:---|:---|:---|
| 127.2% | 보수적 목표 | 1차 익절 (TP1) |
| 161.8% | 중립적 목표 | 2차 익절 (TP2) |
| 261.8% | 공격적 목표 | 3차 익절 (TP3) |

### 3.2 엘리엇 파동 목표가

**파동 3 목표** (파동 1 대비):
- 보수적: 파동 1 x 1.618
- 중립적: 파동 1 x 2.0
- 공격적: 파동 1 x 2.618

### 3.3 ATR 기반 목표가

    목표가 = 진입가 + (ATR x 승수)
    - TP1: ATR x 2.0 (손익비 1:2)
    - TP2: ATR x 3.0 (손익비 1:3)
    - TP3: ATR x 4.0 (손익비 1:4)

## 6. 고급 기법

### 6.1 베팅 켈리 공식 (Kelly Criterion)

    f* = (bp - q) / b
    
    f* = 자본의 투자 비율
    b = 수익/손실 비율
    p = 승률
    q = 패률 (1-p)

**예시**: 승률 40%, 손익비 1:3
- f* = (3x0.4 - 0.6) / 3 = 0.2 (자본의 20%)

### 6.2 포지션 사이징

**퍼센트 리스크법**:

    리스크 금액 = 계좌 잔고 x 리스크 % (예: 1%)
    포지션 크기 = 리스크 금액 / (진입가 - 손절가)
