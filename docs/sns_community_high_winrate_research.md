## ⚠️ **RSI + 볼린저밴드 조합 문제점**

### **TradingRush 백테스트 결과 (100회 거래)**
| 항목 | 결과 |
|:---|:---|
| **승률** | **35%** (매우 낮음) |
| **R:R** | 1.5:1 |
| **평가** | 두 지표 조합이 오히려 약화 |

### **원인 분석**
```
❌ RSI + BB 동시 조건 = 너무 엄격
   - RSI 과매도 + BB 하단 동시 충족 드묾
   - 진입 시점 놓침 (미스된 반등)
   
❌ 상관관계 높음
   - 두 지표 모두 평균회귀 성격
   - 동일한 정보 중복 사용
   
✅ 해결책
   - RSI 단독 사용 + BB는 보조적 확인만
   - 또는 BB + 추세 지표 (EMA, MACD) 조합
```
