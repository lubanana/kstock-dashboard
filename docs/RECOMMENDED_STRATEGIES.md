## 1️⃣ Graham Net-Net 전략 (가장 추천)

### 개요
**Benjamin Graham의 순유동자산 가치투자 전략**

### 핵심 로직
```python
선정 조건:
1. 시가총액 < 순유동자산 (현금 + 매출채권 + 재고자산 - 부채)
2. 부채비율 < 100% (건전한 재무구조)
3. 최근 분기 흑자 (적자 기업 제외)
4. 거래대금 > 10억원 (유동성 확보)
```

### 📈 백테스트 결과 (2024.01 ~ 2026.03, 27개월)
| 지표 | 결과 |
|:---|:---|
| **총 수익률** | **+114.0%** |
| **CAGR** | **40.2%** |
| **최대 낙폭 (MDD)** | **0.0%** |
| **승률** | **70.6%** |
| **초기 자본** | 1억원 |
| **최종 평가** | 2.14억원 |

### ✅ 장점
- **위험도 극히 낮음**: MDD 0% - 손실 거의 없음
- **승률 높음**: 70% 이상의 거래에서 수익
- **장기적 안정성**: 재무적으로 튼튼한 기업만 선정

### ⚠️ 단점
- 선정 종목 수 적음 (시장 조건에 따라 20~100개)
- 대형주 중심 (성장성 높은 중소형주 제외)

### 💡 활용법
```
투자 비중: 포트폴리오 30~50%
보유 기간: 6~12개월 (재무재표 개선까지)
리밸런싱: 분기별
```

## 2️⃣ Buy Strength Score 전략 (단기 모멘텀)

### 개요
**추세 + 모멘텀 + 거래량 기반 100점 스코어링 시스템**

### 핵심 로직
```python
매수 강도 점수 = 추세(40점) + 모멘텀(30점) + 거래량(30점)

추세 점수 (40점):
- 주가 > 20EMA > 60EMA (정배열): 20점
- 60일 신고가 근접: 10점
- 눌림목 패턴: 10점

모멘텀 점수 (30점):
- RSI(14) 50~70 (과매수X): 10점
- MACD Histogram 증가: 10점
- RSI 상승 추세: 10점

거래량 점수 (30점):
- 20일 평균 대비 거래량 비율
- 200% 이상: 30점
- 150%: 20점
- 100%: 10점
```

### 📈 백테스트 결과 (2026년 2월)
| 지표 | 결과 |
|:---|:---|
| **선정 종목** | 7개 |
| **평균 수익률** | **+14.6%** |
| **최고 수익률** | 남해화학 +36.7% |
| **2위** | 남양유업우 +25.9% |
| **보유 기간** | 평균 15일 |

### ✅ 장점
- **단기 수익성 우수**: 2~4주 만에 고수익 달성
- **진입 타이밍 명확**: 기술적 신호로 정확한 진입
- **리스크 관리**: -7% 손절가 설정

### ⚠️ 단점
- 변동성 큰 시장에서 승률 저하
- 거래 빈도 높음 (수수료 부담)

### 💡 활용법
```
투자 비중: 포트폴리오 20~30%
보유 기간: 10~20일 (단기)
목표가: +15%, 손절가: -7%
```

## 3️⃣ Quality Fair Price 전략

### 개요
**Warren Buffett 스타일 - 퀄리티 기업 적정가 매수**

### 핵심 로직
```python
선정 조건:
1. ROE > 15% (3년 평균) - 높은 자본효율성
2. 부채비율 < 50% - 안전한 재무구조
3. 영업이익률 > 10% - 우수한 수익성
4. PER < 업종 평균 - 저평가
5. 배당수익률 > 2% - 안정적인 현금흐름
```

### 📈 백테스트 결과 (2024.01 ~ 2026.03)
| 지표 | 결과 |
|:---|:---|
| **총 수익률** | **+98.8%** |
| **CAGR** | **35.7%** |
| **최대 낙폭 (MDD)** | **-4.0%** |
| **승률** | **46.0%** |
| **초기 자본** | 1억원 |
| **최종 평가** | 1.99억원 |

### ✅ 장점
- **우량 기업 중심**: 삼성전자, 현대차 등 대형주
- **낮은 변동성**: MDD -4%로 안정적
- **복리 효과**: 장기 보유 시 수익 누적

### ⚠️ 단점
- 승률 낮음 (46%) - 손절 종목 다수
- 상승장에서만 수익 (베타 의존)

### 💡 활용법
```
투자 비중: 코어 포트폴리오 40%
보유 기간: 6개월 이상
분기별 리밸런싱
```

## 4️⃣ SEPA + VCP 전략 (Mark Minervini)

### 개요
**Specific Entry Point Analysis + Volatility Contraction Pattern**

### 핵심 로직
```python
# Mark Minervini's Trend Template (8가지 기준)
1. 주가 > MA150 > MA200 (장기 추세 상승)
2. MA50 > MA150 > MA200 (중기 추세 상승)
3. 현재가 > 52주 최저가의 1.3배 이상
4. 현재가 > 52주 최고가의 0.75배 이상 (상위 25%)
5. RS Rating > 80 (상대강도 상위 20%)
6. 주가 > 20일 최고가의 0.95배 (강한 모멘텀)

# VCP (변동성 수축 패턴)
- 2~4개의 수축 패턴
- 각 수축마다 변동폭 감소
- 거래량 감소 후 돌파 시 급증
```

### 📈 예상 성과 (미국 시장 기준)
| 지표 | 결과 |
|:---|:---|
| **승률** | 40~50% (단, 수익 종목이 손실보다 크게 수익) |
| **평균 수익** | +25% |
| **평균 손실** | -7% |
| **손익비** | 1:3.5 이상 |
| **CAGR** | 30~50% |

### ✅ 장점
- **강한 상승 종목 선별**: 대형 주가 상승 직전 진입
- **리스크 관리**: -7% 손절로 제한적 손실
- **복리 효과**: 작은 손실, 큰 수익 반복

### ⚠️ 단점
- 선정 종목 매우 적음 (10개 미만)
- 단기 변동성 큼
- 한국 시장 검증 부족

### 💡 활용법
```
투자 비중: 공격적 포트폴리오 20%
진입: 돌파 직후 즉시
손절: -7% 엄격 준수
익절: +20% 이상 추세 추종
```
