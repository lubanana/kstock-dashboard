# 📊 한국 주식 스캐너 연구 보고서

## 개요

두 가지 유형의 매수 기회를 포착하는 스캐너 시스템 개발

1. **실적/모멘텀 탄탄 + 과매도 종목** (Quality Oversold)
   - 장기 투자 적합
   - 리스크 대비 수익률 우수
   
2. **단기 급등 가능성 종목** (Short-term Surge)
   - 단기 매매 적합
   - 모멘텀 기반 고수익 추구

---

## 1️⃣ 실적/모멘텀 탄탄 + 과매도 스캐너

### 핵심 개념

**"좋은 회사가 싸게 팔릴 때 사라"** - 워런 버핏의 가치투자 철학

### 선정 기준

#### A. 과매도 조건 (Oversold) - 40%

| 지표 | 조건 | 가중치 |
|:---|:---|:---:|
| RSI(14) | < 30: 40점 / < 40: 25점 / < 50: 10점 | 40% |
| Stochastic K | < 20: 30점 / < 30: 15점 | 30% |
| 볼린저밴드 위치 | < 0.1: 30점 / < 0.2: 15점 | 30% |

**근거:**
- RSI < 30: 과매도 구간, 반등 확률 높음
- Stochastic < 20: 단기 과매도 신호
- BB 하단 터치: 지지선 근접

#### B. 가격 하락 (Price Decline) - 30%

| 조건 | 설명 |
|:---|:---|
| 52주 최고가 대비 | -15% 이상 하락 |
| 20일 이동평균 | 하회 중 |
| 하락률 점수 | 최대 30% 하락 시 만점 |

**근거:**
- 52주 고점 대비 -15%: 충분한 조정
- 기관/외국인 매도 물량 소진 시점

#### C. 거래량 (Volume) - 20%

| 조건 | 설명 |
|:---|:---|
| 20일 평균 대비 | 0.8배~1.5배 (정상 수준) |
| 급격한 축소 아님 | 매도 피로도 감소 확인 |

**근거:**
- 거래량 급감 = 매도 세력 소진
- 적정 거래량 = 유동성 확보

#### D. 추세 (Trend) - 10%

| 조건 | 설명 |
|:---|:---|
| EMA20 > EMA60 | 중기 추세 상승 |
| 점수 | 10점 부여 |

### 종합 점수 산출

```
총점 = 과매도점수×0.4 + 하락점수 + 거래량점수 + 추세점수

선정 기준:
- 과매도 점수 ≥ 50점
- 52주 고점 대비 -15% 이상
- 거래량 0.8배+
- 총점 60점+
```

### 매수 시그널

| 등급 | 조건 | 매수 전략 |
|:---|:---|:---|
| STRONG_BUY | 총점 ≥ 70 | 적극 매수 |
| BUY | 총점 60-69 | 분할 매수 |

### 리스크 관리

```
손절가: 진입가 -7%
목표가: 진입가 +15% (1차) / +25% (2차)
보유기간: 2-4주
```

## 2️⃣ 단기 급등 가능성 스캐너

### 핵심 개념

**"추세의 시작을 가장 빠르게 포착하라"** - 리버모어의 모멘텀 투자

### 선정 기준

#### A. 돌파 (Breakout) - 35%

| 지표 | 조건 | 점수 |
|:---|:---|:---:|
| 20일 고점 돌파 | 현재가 > 20일 최고가 × 0.98 | 30점 |
| 20일선 상회 | 현재가 > EMA20 | 20점 |
| 단기 골든크로스 | EMA5 > EMA20 | 15점 |

**근거:**
- 20일 고점 돌파 = 새로운 상승 추세 시작
- 이동평균선 정배열 = 강한 상승 동력

#### B. 모멘텀 (Momentum) - 35%

| 지표 | 조건 | 점수 |
|:---|:---|:---:|
| RSI | 50-70 (건강한 모멘텀) | 25점 |
| MACD | 골든크로스 | 25점 |
| MACD 히스토그램 | 증가세 | 15점 |
| ADX | > 25 (강한 추세) | 15점 |
| DI+ > DI- | 상승 DI 우세 | 10점 |

**근거:**
- RSI 50-70: 과매수 아닌 강한 모멘텀
- MACD 골든크로스: 단기 상승 전환
- ADX > 25: 추세의 강도 확인

#### C. 거래량 폭발 (Volume Burst) - 30%

| 조건 | 설명 | 점수 |
|:---|:---|:---:|
| 20일 평균 대비 2.0배+ | 강한 매수세 유입 | 35점 |
| 20일 평균 대비 1.5배+ | 매수세 증가 | 25점 |
| 20일 평균 대비 1.2배+ | 거래량 증가 | 15점 |
| 3일 연속 증가 | 지속적 유입 | 15점 |

**근거:**
- 거래량 2배 = 기관/외국인 대량 매수
- 연속 증가 = 추세 지속 가능성

#### D. 캔들 패턴 - 10%

| 패턴 | 설명 | 점수 |
|:---|:---|:---:|
| 양봉 | 종가 > 시가 | 10점 |
| 장대양봉 | 몸통 > 전체 범위 60% | 10점 |
| 아래꼬리 | 하방 지지 확인 | 5점 |

**근거:**
- 양봉 = 매수 우세
- 아래꼬리 = 저점 매수세

### 필수 조건

```
1. 거래량 1.5배+ (필수)
2. 거래대금 10억원+ (필수)
3. 양봉 (필수)
4. RSI < 70 (과매수 아님)
5. 총점 60점+
```

### 매수 시그널

| 등급 | 조건 | 매수 전략 |
|:---|:---|:---|
| STRONG_BUY | 총점 ≥ 80 | 적극 추격 매수 |
| BUY | 총점 60-79 | 신중 추격 매수 |

### 리스크 관리

```
손절가: 진입가 -5% (단기)
목표가: 진입가 +10% (1차) / +20% (2차)
보유기간: 1-3일 (초단기) / 1주 (단기)
```
