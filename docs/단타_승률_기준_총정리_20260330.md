## 📈 기술적 지표 기반 전략

### 1️⃣ RSI + 볼린저밴드 조합 (추천 ⭐⭐⭐)

#### 진입 조건
```
[매수 진입]
✓ RSI 30 이하 (과매도)
✓ 주가가 볼린저밴드 하단선 아래에서 위로 강하게 돌파
✓ 거래량 평균 대비 1.5배 이상 증가

[매도 진입]
✓ RSI 70 이상 (과매수)
✓ 주가가 볼린저밴드 상단선 위에서 아래로 강하게 하향 돌파
✓ 거래량 평균 대비 1.5배 이상 증가
```

#### 볼린저밴드 설정
| 종목 유형 | 기간 | 표준편차 |
|:---|:---:|:---:|
| 대형주 | 60일 | 2σ |
| 중소형주 | 40일 | 2σ |
| 바이오/변동성 높음 | 80일 | 2.5σ |
| 가상화폐 | 30일 | 3σ |

#### 승률 향상 팁
- **60일 BB 하단 터치 + RSI 30 이하** 동시 발생 시 매수 성공률 크게 증가
- 단순히 닿는 것이 아니라 **강하게 돌파**할 때 진입
- BB 수축 후 확장 구간에서 효과 극대화

### 2️⃣ MACD + 볼린저밴드 조합

#### 진입 조건
```
✓ MACD 선이 시그널선 아래에서 위로 골든크로스
✓ 동시에 60일 볼린저밴드 중심선(MA60) 돌파
✓ 거래량 증가 확인
```

#### 해석
| 상황 | 신호 |
|:---|:---:|
| MACD > 시그널선 (베이스라인 상향) | 🟢 매수 |
| MACD < 시그널선 (베이스라인 하향) | 🔴 매도 |
| MACD가 시그널선 위아래로 크게 진동 | ⚠️ 변동성 큼 (관망) |

---

### 3️⃣ 볼린저 스퀴즈 (Breakout 전략)

#### 개념
- BB 폭이 좁아지는 구간 = 변동성 증가 직전
- **수축(Squeeze) → 확장(Expansion)** 패턴 포착

#### 진입 조건
```
✓ BB 상단/하단 간격이 최근 20일 중 최소
✓ 거래량이 평균의 70% 이하로 감소 (수축 확인)
✓ 이후 거래량 급증 + 방향성 발생 시 진입
```

---

### 4️⃣ 볼륨 프로필 (고급 전략)

#### 핵심 개념
- **POC (Point of Control):** 가장 많은 거래량이 발생한 가격
- **Value Area:** 전체 거래량의 70%가 형성된 가격 범위
- **High Value Area / Low Value Area:** 가치 영역의 상하단

#### 진입 전략
```
✓ 가격이 POC 근처에서 지지/저항 확인
✓ Value Area 돌파 시 추세 추종 진입
✓ 거래량이 적은 구간(공백)은 빠른 이동 예상
```

## 💡 핵심 인사이트

### 1. "시간 = 리스크"
> 한국 시장은 순환장 구조. 장기 보유는 리스크 누적.
> 단타는 짧은 시간 안에 리스크 통제 가능.

### 2. "고승률 > 대박"
> 소수의 대박보다 지속적이고 일관된 수익이 핵심.
> 80% 승률을 목표로 기계적 반복 실행.

### 3. "지표 교차 확인"
> 하나의 지표에 의존하지 말고 최소 2~3개 지표로 교차 검증.
> 단순히 밴드에 닿았다고 매매하지 말 것.

### 4. "추세가 우선"
> 모든 지표는 추세를 따름.
> 강한 상승 추세에서는 과매수가 계속될 수 있음.
> 강한 하락 추세에서는 과매도가 계속될 수 있음.
