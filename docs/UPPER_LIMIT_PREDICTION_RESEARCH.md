## 3. 핵심 필터 조건

### 3.1 필수 조건 (모두 충족)

✅ **거래량 4배 이상**
- 대량의 자금 유입 확인
- 세력/기관 매집 신호

✅ **연속 상승 3일+**
- 모멘텀 확인
- 추세 전환 신호

✅ **저가주 (5만원 이하)**
- 상한가 도달 시 심리적 효과
- 개인 투자자 참여 용이

### 3.2 우대 조건 (있으면 좋음)

⭐ **테마/호재**
- 정책 발표 (건설, 반도체 지원)
- 수주 공시
- 신제품 출시

⭐ **차트 패턴**
- 쌍바닥 돌파
- 삼각형 수렴 후 돌파
- 거래량 급증 + 주가 돌파

⭐ **수급**
- 외국인/기관 순매수
- 공매도 급감
- 대주주 지분 증가

## 4. 상한가 예측 체크리스트

### 매일 장 마감 후 확인

```
□ 1. 거래량 급증 종목 선별 (전일 대비 4배+)
□ 2. 당일 상승률 확인 (5% 이상)
□ 3. 연속 상승 일수 확인 (3일+)
□ 4. 저가주 필터 적용 (5만원 이하)
□ 5. 테마/호재 확인
□ 6. 점수 계산 (60점+ 집중)
□ 7. 차트 패턴 확인
□ 8. 다음날 감시대상 선정
```

### 다음날 매매 전략

**시초가 매매**:
- S급(80점+) 종목: 시초가 진입
- A급(65점+) 종목: 갭 상승 시 진입

**중간 매매**:
- 10% 돌파 시 추격 매수
- 거래량 지속 확인

**청산 전략**:
- 상한가 도달: 50% 익절, 50% 다음날 감시
- 20% 이상: 전량 익절
- -5% 손실: 손절
