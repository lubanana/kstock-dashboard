## 💡 왜 저가 진입이 우수한가?

### 1. 변동성 활용
- 다음날 하락 시 저점에서 매수
- 평균 진입가가 시가 대비 **-1.5%~-2.5%** 저렴
- 같은 종목이라도 더 유리한 가격 진입

### 2. 리스크 감소
- 낮은 진입가 = 더 넓은 손절 여유
- 변동성 흡수 능력 향상
- 심리적 안정감

### 3. 수익률 극대화
- 저점 매수로 상승 시 수익률 증가
- R:R 비율 유지하면서 승률 향상

## 📈 실전 적용 가이드

### 조건 설정

```python
# 스캔 조건
ichimoku_params = {"tenkan": 5, "kijun": 10}  # 단기
min_price = 2000  # 2,000원 이상
liquid_top_n = 300  # 거래대금 상위 300종목

# 매매 조건
entry_type = "next_low"  # 다음날 저가 진입
stop_loss = -5%          # 손절
take_profit = +10%       # 익절
holding_days = 5         # 최대 보유
```

### 실행 방법

**1. 장중 모니터링**
- 전일 TK_CROSS 발생 종목 리스트업
- 당일 저가 도달 시 지정가 매수 주문

**2. 자동 주문 설정**
```
전일 종목 중 TK_CROSS 발생 시:
→ 당일 지정가: 전일 종가 × 0.985 (1.5% 아래)
→ 손절가: 지정가 × 0.95
→ 익절가: 지정가 × 1.10
```

**3. 리스크 관리**
- 1종목당 자본의 2~3% 배분
- 일일 최대 5종목 진입 제한
- 연속 손절 시 당일 휴식
