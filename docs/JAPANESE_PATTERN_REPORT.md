## ⚠️ 발견된 문제점

### 1. 패턴 감지 로직 문제
- **삼산/삼천**: 거의 모든 종목에 동시에 감지됨
- **원인**: 단순 분산 기준으로 인한 과다 감지
- **해결 필요**: 피크/트로우 탐색 알고리즘 개선 필요

### 2. 신호 신뢰도 부족
- TK_CROSS만으로는 예측력 부족
- 전통적인 일목균형표 해석이 현대 시장에서는 약함

### 3. 실행 지연
- 스캔 시간: 500종목당 ~2분 소요
- 전체 시장 스캔 시 비실용적
