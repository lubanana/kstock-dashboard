## 🎯 분석 및 평가

### 종합 평가: ❌❌ **사용 금지**

| 평가 항목 | 등급 | 설명 |
|:---|:---:|:---|
| **승률** | F (18-19%) | 5건 중 4건 손실 |
| **수익률** | F (-2.3~2.5%) | 기대값 마이너스 |
| **리스크관리** | C | 익절/손절 비율 양호하나 승률 저하 |
| **실용성** | D | 80% 손절은 실행 불가 |

### 핵심 문제점

**1. 눌림목 = 추가 하락의 시작**
- 거래대금 급증 후 다음날 하락은 단기 조정이 아닌 **추가 하락의 신호**
- 80%가 손절에 걸리는 것은 통계적으로 유의미

**2. 테마주 특성**
- 급등 후 급락이 반복되는 종목 특성
- "눌림목"이라는 개념이 테마주에는 적용되지 않음

**3. 손익비 불균형**
- 승률 19% × 평균 수익 15% = 2.85
- 패배율 81% × 평균 손실 5% = 4.05
- 기대값 = -1.2 (손실)
