# WORK_RULES.md - 작업 원칙

## 🚫 절대 수정하지 않는 항목

### 1. 백테스트 기반 기준
- **70점 이상**: 승률 80% (적극 매수)
- **60-69점**: 승률 40% (참고용)
- **60점 미만**: 제외

> ⚠️ **중요**: 사용자가 명시적으로 지시하지 않는 한 절대 수정 금지

### 2. 보유 기간
- **3일 이상 보유**: 승률 64.3%
- **1일 스윙**: 승률 45.2% (비권장)

### 3. 리스크 관리
- **개별 종목**: 최대 3% 포지션
- **손절가**: -3%
- **익절가**: +5%

---

## ✅ 수정 가능한 항목 (사용자 요청 시)

- 스캔 대상 종목 수 (--top N)
- 출력 형식 (HTML, JSON, 테이블)
- 리포트 생성 여부 (--export)
- 알림 설정

---

## 📝 확인 절차

기준 변경 필요 시:
1. 사용자에게 변경 요청 확인
2. 백테스트 데이터 검증
3. 변경 사유 문서화
4. 사용자 승인 후 수정

---

*이 원칙은 사용자의 명시적 지시 없이는 변경되지 않습니다.*
