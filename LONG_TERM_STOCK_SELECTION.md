# 📈 장기 고수익 종목 발굴 방법론

## 개요
장기투자(3년~10년+)에서 높은 수익을 기대할 수 있는 종목을 발굴하는 체계적인 방법론

---

## 1️⃣ 가치투자 (Value Investing) 기준

### 워런 버핏의 5가지 원칙

| 원칙 | 설명 | 체크포인트 |
|:---|:---|:---|
| **내재가치 대비 저평가** | "1달러짜리를 50센트에 사라" | PER < 15, PBR < 1.5 |
| **이해 가능한 사업** | 복잡하지 않고 설명 가능한 비즈니스 | 5분 안에 사업 모델 설명 가능 |
| **경제적 해자(Moat)** | 지속적인 경쟁 우위 | 브랜드/특허/규모의 경제/네트워크 효과 |
| **경영진 역량** | 정직성과 자본운용 능력 | CEO 주주서한, 배당 정책 확인 |
| **시장 공포 활용** | "남들이 두려워할 때 욕심내" | 위기 시 매수 기회 포착 |

### 핵심 재무 지표

```python
# 가치투자 필터 조건
value_criteria = {
    'PER': '< 15',           # 주가수익비율
    'PBR': '< 1.5',          # 주가순자산비율  
    'ROE': '> 15%',          # 자기자본이익률
    '부채비율': '< 100%',     # 재무안정성
    'FCF': '양수',            # 잉여현금흐름
    '배당수익률': '2~4%',     # 배당 안정성
}
```

### 경제적 해자(Moat) 유형

| 해자 유형 | 예시 | 확인 방법 |
|:---|:---|:---|
| **브랜드 해자** | 코카콜라, 애플 | 브랜드 파워, 가격 결정권 |
| **비용 우위** | 월마트, 코스트코 | 원가 구조, 규모의 경제 |
| **네트워크 효과** | 아메리칸 익스프레스 | 사용자 증가 → 가치 증가 |
| **규제/특허** | 제약사, 반도체 | 특허 보유, 진입장벽 |
| **고착성** | 마이크로소프트, 애플 | 전환비용, 생태계 잠금 |

## 2️⃣ 성장투자 (Growth Investing) 기준

### 성장주 선정 공식

```
성장성 점수 = (Y+1년 영업이익 - MAX(Y-1년, Y-2년 영업이익)) / 시가총액
```

### 핵심 필터

| 지표 | 기준 | 의미 |
|:---|:---|:---|
| **영업이익 성장률** | > 20% | 지속적인 이익 증가 |
| **5년 영익 성장** | > 15% | 장기 성장 추세 |
| **매출 성장률** | > 20% | 시장 확대 확인 |
| **PEG Ratio** | < 1 | 성장 대비 저평가 |
| **부채/자산** | < 0.25 | 재무 건전성 |
| **부채비율** | < 0.2 | 낮은 레버리지 |

### 유망 산업 (2025~2030)

| 산업 | 연 성장률 | 대표 섹터 |
|:---|:---:|:---|
| 인공지능(AI) | +30% | 반도체, 클라우드, AI 서비스 |
| 2차전지 | +25% | 배터리 소재, 장비 |
| 클라우드 | +20% | SaaS, 데이터센터 |
| 바이오/헬스케어 | +18% | 신약, 의료기기 |
| 전기차 | +22% | 배터리, 부품, 충전 |
| 우주산업 | +15% | 위성, 발체 |

```python
    '자산확대': 'Gross block > 1.2배',
    '현금흐름': 'FCF 3년 연속 플러스',
}
```

### DCF(현금흐름 할인법) 간편 계산

```python
def simple_dcf(fcf_growth_5y=0.15, terminal_growth=0.03, discount_rate=0.10):
    """
    fcf_growth_5y: 5년간 FCF 성장률
    terminal_growth: 영구 성장률
    discount_rate: 할인율
    """
    # 5년 FCF 합계
    fcf_sum = sum([(1 + fcf_growth_5y) ** i / (1 + discount_rate) ** i 
                   for i in range(1, 6)])
    
    # Terminal Value
    terminal_value = ((1 + fcf_growth_5y) ** 5 * (1 + terminal_growth)) / \
                     (discount_rate - terminal_growth)
    terminal_pv = terminal_value / (1 + discount_rate) ** 5
    
    return fcf_sum + terminal_pv

# 내재가치 > 현재 주가 → 저평가
```

## 4️⃣ 장기 추세 추종 (Long-term Trend Following)

### 장기 보유 전략

| 전략 | 보유기간 | 진입 조건 | 특징 |
|:---|:---:|:---|:---|
| **GARP** | 3~5년 | 성장 + 가치 결합 | 합리적 가격의 성장주 |
| **퀄리티** | 5년+ | ROE > 20%, 부채 낮음 | 우량주 중심 |
| **모멘텀** | 1~3년 | 12개월 수익률 상위 | 추세 유지 종목 |
| **배당성장** | 10년+ | 배당 10년 연속 증가 | 안정적 소득 |

## 6️⃣ 실전 포트폴리오 구성

### 장기투자 포트폴리오 (5년+ 보유)

```
[코어 (60%)]
├── 가치주 30% - PER < 15, 배당 3%+
├── 퀄리티 30% - ROE > 20%, 부채 < 50%

[성장 (30%)]
├── GARP 15% - 성장 + 가치 결합
├── 모멘텀 15% - 추세 상위 종목

[대안 (10%)]
└── 배당성장 10% - 10년+ 배당 증가
```

### 리밸런싱 전략

| 주기 | 작업 |
|:---|:---|
| **연 1회** | 종목별 실적 업데이트, 점수 재계산 |
| **분기별** | 해자 변화 확인, 경쟁사 동향 체크 |
| **월간** | 추세 변화, 거래량 확인 |
| **실시간** | 악재 발생 시 긴급 점검 |

## 💡 핵심 인사이트

1. **시간이 아군** - 장기투자는 복리의 힘을 활용
2. **해자가 생명** - 경쟁 우위가 없으면 장기 보유 불가
3. **가격 중요** - 좋은 기업도 비싸게 사면 손해
4. **인내심 필수** - 10년 보유가 어렵다면 장기투자 아님
5. **집중 투자** - 10개 이하 종목 집중이 유리
